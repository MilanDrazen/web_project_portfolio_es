# Nombre Proyecto:

Portafolio de Milán Drazen, Desarrollador Web

# Descripción:

Este proyecto funciona como una página de muestra de los proyectos realizados por el desarrollador web, se presentan en un formato más amigable para los clientes que no manejan GitHub.

# Tecnologías y Herramientas:

Para el desarrollo de esta página se utilizaron los lenguajes HTML y CSS, además se siguió la lógica de BEM (Block, Element, Modifier) para la estructuración de código, también se utilizó un poco de Atomic CSS para volver más recursivos algunos elementos, además se usó pensamiento DRY (Don't Repeat Yourself) para dar un orden lógico al código.
